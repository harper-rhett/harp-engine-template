# HarpEngine Template
This is a template for the game framework [HarpEngine](https://github.com/harper-rhett/harp-engine). It has a basic example to help you get started on a new project. Install the zip file from [GitHub Releases](https://github.com/harper-rhett/harp-engine-template/releases) and then drop the file into `C:\Users\%USERNAME%\Documents\Visual Studio 2022\Templates\ProjectTemplates`.
